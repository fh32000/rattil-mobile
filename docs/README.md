# Warattilhu (ورتِّله) Documentation

**Warattilhu** is a Flutter-based Quran recitation app focused on Juz Amma (جزء عمّ, Surahs 78-114) and Arabic alphabet pronunciation (Tajweed). It features background audio playback, lock-screen controls, favorites, playlists, and a rich RTL Arabic UI.

## App at a Glance

```
User clicks surah → GoRouter navigates → Provider loads audio
→ QuranAudioHandler (BaseAudioHandler + AudioPlayer)
→ Just Audio plays MP3 from assets/audio/juz_amma/
→ UI rebuilds via StreamProvider
→ Position saved to Hive every 3s
```
